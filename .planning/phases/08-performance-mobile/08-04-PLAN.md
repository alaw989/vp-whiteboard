  <name>Replace touch events with pointer events for drawing</name>
  <files>components/whiteboard/WhiteboardCanvas.vue</files>
  <action>
    Replace touch event handlers with pointer events in WhiteboardCanvas.vue:

    1. Add new pointer event handlers:
       ```typescript
       function handlePointerDown(event: any) {
         // Get pointer type: 'mouse', 'pen', 'touch'
         const pointerType = event.evt.pointerType || 'mouse'

         // Handle pressure for pen/stylus (0-1 range, default 0.5)
         const pressure = event.evt.pressure || 0.5

         // Call existing mouse handler
         handleMouseDown(event)
       }

       function handlePointerMove(event: any) {
         handleMouseMove(event)
       }

       function handlePointerUp(event: any) {
         handleMouseUp(event)
       }

       function handlePointerLeave(event: any) {
         handleMouseUp(event) // Treat leave as up
       }
       ```

    2. Update v-stage bindings:
       - Add: @pointerdown="handlePointerDown"
       - Add: @pointermove="handlePointerMove"
       - Add: @pointerup="handlePointerUp"
       - Add: @pointerleave="handlePointerLeave"
       - Remove: @touchstart, @touchmove, @touchend (now redundant)

    3. Keep @mousedown, @mousemove, @mouseup, @mouseleave for desktop
       - Pointer events work for mouse too, but keeping both ensures compatibility

    4. Add pressure support to stroke drawing:
       - Modify handleMouseDown to capture pressure from pointer event
       - Store currentPressure in a ref
       - Use pressure in currentStrokePoints: [x, y, pressure]

    5. Add CSS touch-action to whiteboard-container:
       ```css
       .whiteboard-container {
         touch-action: none; /* Prevent browser gestures */
       }
       ```
       This prevents scrolling and zooming while drawing.

    6. Update getPointerPos to handle pointer coordinates correctly

    Do NOT modify desktop mouse behavior or tool selection logic.
  </action>
