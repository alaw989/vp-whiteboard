<task type="auto">
  <name>Implement two-finger pan gesture in WhiteboardCanvas</name>
  <files>components/whiteboard/WhiteboardCanvas.vue</files>
  <action>
    Add two-finger pan gesture handling to WhiteboardCanvas.vue:

    1. Add gestureState ref:
       ```typescript
       const gestureState = ref({
         isPanning: false,
         initialPositions: [] as Array<{x: number, y: number}>,
         lastViewport: { x: 0, y: 0, zoom: 1 },
       })
       ```

    2. Modify handleTouchStart:
       - Check event.evt.touches.length
       - If touches.length === 2: enter pan mode
         - Set gestureState.value.isPanning = true
         - Store both touch positions in initialPositions
         - Store current viewport in lastViewport
         - Call event.evt.preventDefault()
         - Return (don't call handleMouseDown)
       - If touches.length === 1: proceed to handleMouseDown (drawing)

    3. Modify handleTouchMove:
       - If gestureState.value.isPanning && touches.length === 2:
         - Calculate delta from first finger movement
         - Update viewport: viewport.value.x = lastViewport.x + deltaX
         - Update viewport: viewport.value.y = lastViewport.y + deltaY
         - Call event.evt.preventDefault()
         - Return (don't call handleMouseMove)
       - Otherwise: proceed to handleMouseMove

    4. Modify handleTouchEnd:
       - If touches.length < 2: exit pan mode
         - Set gestureState.value.isPanning = false
       - Proceed to handleMouseUp

    5. Add touch-action CSS to prevent browser gestures:
       - Add style block with .whiteboard-container { touch-action: none; }
       - This prevents pinch-zoom and scroll

    6. Import setViewportDirect from useViewport for efficient updates

    Do NOT modify mouse event handlers or desktop behavior.
  </action>
  <verify>
    1. Two-finger touch starts pan mode (no drawing)
    2. Dragging two fingers pans the canvas
    3. Releasing second finger ends pan mode
    4. Single-finger touch still draws normally
    5. Browser pinch-zoom is prevented
  </verify>
  <done>
    Two-finger pan gesture works for canvas navigation while single-touch drawing remains functional.
  </done>
