<objective>
Add scale setting infrastructure for converting canvas pixels to real-world measurements. Users can define a scale ratio (e.g., "1 inch = 10 feet") via a floating dialog, and the current scale is always visible in a corner badge. Scale state persists per-document and syncs via Yjs.

Purpose: Without a defined scale, measurements would only show pixel values. Setting the scale ratio enables real-world unit display (inches/feet) for all measurement tools.

Output: useScale composable for state management, ScaleBadge component for always-visible display, ScaleToolPalette component for scale input dialog, ScaleState types.
</objective>

       - displayFormat: ComputedRef<string> - e.g., "1\" = 10'" or "No scale set"
       - pixelsPerInch: ComputedRef<number> - defaults to 96 if no scale

    3. Core functions:
       - setScale(drawingUnits: number, drawingUnit: 'inches', realWorldUnits: number, realWorldUnit: 'feet' | 'inches'): void
         * Calculates pixelsPerInch from input ratio
         * Generates label like "1\" = 10'" or "1\" = 100'"
         * Stores in yMeta with userId and timestamp
         * Pattern from useCollaborativeCanvas syncViewport:

       ```typescript
       function setScale(pixelsPerInch: number, unit: 'inches' | 'feet') {
         ydoc.transact(() => {
           yMeta.set('scale', {
             pixelsPerInch,
             unit,
             label: generateLabel(pixelsPerInch, unit),
             lastUpdatedBy: userId,
             timestamp: Date.now()
           })
         }, userId)
       }
       ```

       - getScale(): ScaleState | null - reads from yMeta
       - observeScale(callback: (scale: ScaleState) => void): () => void - observe remote changes

    4. Unit conversion helpers (for use by measurement tools):

       - inchesToFeet(inches: number): number
       - formatMeasurement(inches: number, precision: number): string

    5. Per-document scale support (from context decision):
       - Store scale with document ID prefix: `scale:${documentId}`
       - When document switches, call getScale(documentId) to restore

    Follow existing composable pattern from useCollaborativeCanvas.ts for yMeta interaction and conflict avoidance via lastUpdatedBy.
