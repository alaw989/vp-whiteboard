<task type="auto">
  <name>Create useSnapping composable for geometric snap detection</name>
  <files>composables/useSnapping.ts</files>
  <action>
    Create composables/useSnapping.ts:

    1. SnapPoint interface:
       ```typescript
       export interface SnapPoint {
         x: number
         y: number
         type: 'endpoint' | 'midpoint' | 'center' | 'corner' | 'intersection'
         elementId: string
       }
       ```

    2. Core function:
       findSnapPoint(cursor: { x: number, y: number }, elements: CanvasElement[]): SnapPoint | null

       - SNAP_THRESHOLD = 10 pixels (configurable)
       - For each element, extract geometric features:
         * Line: endpoints (start, end)
         * Rectangle: four corners
         * Circle: center point
         * Ellipse: center point
         * Stroke: first and last points
       - For each feature, check distance to cursor using Math.hypot
       - Return nearest point within threshold, or null if none

    3. Throttling:
       - Use @vueuse/core useDebounceFn with 33ms delay (~30fps)
       - Prevents O(n) performance issues on every mousemove

    4. Optional: midpoint calculation for lines (average of start/end)
       - Add midpoints as snap targets for better UX

    Pattern from research section of 07-RESEARCH.md for snap detection algorithm.
  </action>
